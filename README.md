# 🔐 Password Strength Checker (Python)

## 📌 Description
This project is a Python-based Password Strength Checker that evaluates the security of a user’s password using multiple criteria such as length, uppercase and lowercase letters, digits, and special characters. It classifies passwords as Weak, Medium, or Strong and provides suggestions to help users create more secure passwords

- Weak ❌  
- Medium ⚠️  
- Strong ✅  

It also provides helpful suggestions to improve password security.

---

## 🚀 Features
- 🔍 Checks password strength  
- 📏 Minimum length validation (8 characters)  
- 🔠 Detects uppercase & lowercase letters  
- 🔢 Ensures numeric digits  
- 🔣 Validates special characters  
- 💡 Gives improvement suggestions
